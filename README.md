## Installation
Add this line to your application's Gemfile:

```ruby
gem "polivalente"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install polivalente
```

Copy migrations:
```bash
$ rails g polivalente:install
```
